TinyCMS 1.3 Documentation

Thanks for downloading TinyCMS 1.3, we hope you enjoy it.

Installing TinyCMS

To install TinyCMS, upload the files to your server and you're ready to roll. Visit YourSite.com/admin/ and use the default login details. The default username is admin and the default password is pass.

Installing themes/templates

At this time, we're still creating a great guide for themes. In TinyCMS 1.4, we'll be creating even easier ways to make themes. Check out our themes page for some starters.

Changes to file locations

Note for existing themes. We've moved the JS files from yoursite.com/js to yoursite.com/admin/js to clean up the directories. If you're using a theme from our website, ensure the directories are correct or the site may not work correctly.

Notes

TinyCMS 1.3 addresses an annoying issue with the URLs on new installations. Also, the code has been cleaned and comments have been added. Just compare the 1.2 functions file to the 1.3 functions file, 1.2 was a mess!

1.4 will most likely include an easy way to create new themes, it's pretty easy right now if you know how, but for example some useful classes may come in handy, such as.. $build->nav; or $build->content
