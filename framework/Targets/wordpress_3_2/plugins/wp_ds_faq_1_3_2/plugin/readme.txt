== Description ==

WP DS FAQ plugin is just a simple FAQ pages management tool for your web site.

== Installation ==

Download: <a href="http://wp-plugins.diamondsteel.ru/wp-ds-faq/">WP DS FAQ</a>

1. Extract "wp-ds-faq" folder from the archive.
2. Copy "wp-ds-faq" folder to your "plugins" folder. Usually it's "/wp-content/plugins".
3. Go to the "Plugins" section in the administrative panel and "activate" WP DS FAQ

= Usage =

Plugin exchange special key words in the page text for the FAQ created by you.
 
1. To create a FAQ page you need to:
2. Go to plugin settings.
3. Add FAQ "book".
4. Fill it with questions and answers.
5. Copy "short code"
6. Create a page that has "short code" in the blog
