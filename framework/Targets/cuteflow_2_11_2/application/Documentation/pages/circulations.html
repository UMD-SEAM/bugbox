Circulation messages

The list of circulations

To manage the circulations, click on "Circulations" in the menu (left-side). After that you will see a list of all available circulations.


There you can use the following actions:
Starting a new circulation
Deleting the selected circulation
Observing the selected circulation
Put the selected circulation in the archive
Stops the selected circulation. That means that it isn't send to the next user in the list.


New circulation

If you click on "New circulation" you will get a page like the following:


The following data must be provided to describe a circulation:
Circulationname The name of the circulation. This field is needed!
Mailing list Please select a mailing list, that should be used for the circulation. This field is needed!
Attachments You can choose up to 4 files that will be send with the message as an attachment.
Description text The text you can enter here is shown on top of each circulation email
Success message to sender if circulation is complete If this option is checked, the sender is notified if the circulation is completed
Archive successful circulation automaticly If this option is checked, a completed circulation will be archived automatically.

Optionally you can go through two more Steps.
Step 2:

In the second Step you can adapt the mailinglist.

Step 3:

The third step allows you to edit the standardvalues of the fields before sending the circulation.

Observing circulations

For the sender it's important at which station a circulation is currently processed. For that you can click on circulation details and get view on the current status of an circulation.



The circulation archive

Circulation could be moved to the circulation archive. You can move i.e. old, completed circulations to the archive, for getting a more clean view on your currently running ones.
